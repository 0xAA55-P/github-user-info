# Tasque-manager
 - Criei este projeto para praticar consumo de APIS.
 - Retorna algumas informações de um usuario do github com base em seu nome

- **Github API** - Receber informações do usuario ([site](https://docs.github.com/en/rest/users))

# Pré-requisitos
- Antes de começar, certifique-se de que cumpre os seguintes requisitos:
	- Python >= 3.9 (3.11+ recomendado)
	- rich, requests >= Vá para a area de Instalação

## Instalação

- Veja como instalar:

```bash
# Clonar o repositório
git clone https://github.com/0xAA55-P/github-user-info

# Instalar dependências
pip install -r requisitos.txt

# Executar o aplicativo
python run.py
```
